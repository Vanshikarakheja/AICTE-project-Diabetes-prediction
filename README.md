# AICTE-project-Diabetes-prediction
Diabetes Prediction Web App

This project is a Machine Learning web application that predicts whether a person is likely to have diabetes based on medical parameters. The model is trained on the Pima Indians Diabetes Dataset and deployed using Streamlit for an interactive interface.

🚀 Features

Predicts diabetes risk based on health inputs

Simple and interactive web interface

Real-time predictions using a trained ML model

🧠 Model

The prediction model uses SVM classifier trained on features such as glucose level, BMI, blood pressure, age, and insulin level.
